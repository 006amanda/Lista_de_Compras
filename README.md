# 🛒 Lista de Compras com Flask

## 📌 Descrição
Este projeto consiste em uma aplicação web simples desenvolvida com Flask que permite ao usuário criar e gerenciar uma lista de compras.

Os itens adicionados são exibidos em um card de "Comprar" e, ao serem marcados, são automaticamente movidos para o card de "Comprados".

---

## ⚙️ Funcionalidades

- Adicionar novos itens à lista de compras
- Selecionar a quantidade de cada item
- Visualizar itens pendentes
- Marcar e desmarcar itens
- Movimentação automática entre listas
- Interface simples e intuitiva

---

## 🛠️ Tecnologias utilizadas

- Python
- Flask
- HTML5
- CSS3
